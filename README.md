# Template
This is template for Obratnaya.

## Usage
```
$ make help
Help v1.0
Usage: make [args...]
args:
	No args: Install Obratnaya and run Obratnaya.
	install: Install Obratnaya.
	run    : Run Obratnaya.
	license: Display about license info.
	         Need OPT=legalcode/name
	clean  : Uninstall Obratnaya.
```
